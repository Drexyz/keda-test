## Step to Run Tests

1. **Install dependencies**  
   ```bash
   npm install
   ```

2. **Run all test files**  
   ```bash
   npm test
   ```

3. **Run a specific test file**  
   ```bash
   npm test {file-name}
   ```

   Replace `{file-name}` with part of the file name. Example:
   ```bash
   npm test sort
   ```
